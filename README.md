# LearnHub

LearnHub is an Online Learning Platform that helps users learn new skills through online courses.  
It provides flexible, self-paced learning for students and easy course management for teachers.

## Project Description

LearnHub allows students to register, browse courses, enroll, and learn at their own pace.  
Teachers can create and manage courses, while admins control the platform.

## Users

- Student  
- Teacher  
- Admin  

## Features

- User registration and login  
- Browse and enroll in courses  
- Video lectures and study materials  
- Course progress tracking  
- Discussion forums and interaction  
- Certificates after course completion  
- Free and paid courses  

## Scenario Example

A student registers on LearnHub, logs in, browses courses, enrolls in a course, learns through videos and materials, completes the course, and receives a certificate.

## Technologies Used

Frontend:
- React.js
- Bootstrap
- Material UI

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## System Architecture

LearnHub follows a client-server architecture:
- Frontend handles user interface
- Backend handles logic and APIs
- Database stores application data

## Advantages

- Learn anytime and anywhere  
- Easy to use  
- Skill-based learning  

## Future Enhancements

- Mobile application

## Conclusion

LearnHub is a simple and effective online learning platform that helps users enhance their skills through online education.
